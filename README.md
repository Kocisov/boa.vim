# boa.vim
Colorscheme for vim and neovim.
Working only in terminals with 24-bit colour support.

Add this lines to your .vimrc
```
set guicolors
set background=dark
colorscheme monkoose
```

For neovim add this lines to your init.vim(neovim config file)
```
set termguicolors
set background=dark
colorscheme monkoose
```
